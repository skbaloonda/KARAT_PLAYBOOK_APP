Practice Questions

Comprehensive set of practice questions for junior engineering candidates.

Priority Questions

1

Reverse a string in-place (or using language facility).

Answer: Use two-pointer swap from both ends to center for O(n) time and O(1) extra space; or use built-in reverse/slicing for concise solutions.
2

Two-sum: find indices of two numbers that add to target.

Answer: Use a hashmap to store value->index while iterating; for each number check if target - num exists. O(n) time, O(n) space.
3

Detect a cycle in a singly linked list.

Answer: Use Floyd's tortoise and hare (slow/fast pointers). If they meet, a cycle exists. O(n) time, O(1) space.
4

Implement FizzBuzz for numbers 1..N.

Answer: Loop 1..N, print 'Fizz' if divisible by 3, 'Buzz' if divisible by 5, 'FizzBuzz' if both, else the number. O(n) time, O(1) space.
5

Check if a string is an anagram of another.

Answer: Count characters (hashmap or array of 26 for lowercase) for both strings and compare counts; sorting also works but is O(n log n).
6

Perform binary search on a sorted array.

Answer: Repeatedly compare middle element with target and narrow left/right bounds. Implementation can be iterative or recursive. O(log n) time.
7

Merge two sorted arrays into one sorted array.

Answer: Use two-pointer technique from the starts (or ends) of arrays and append smaller element each step. O(n+m) time, O(n+m) space (or in-place when buffer exists).
8

Explain time and space complexity (Big-O) simply.

Answer: Big-O describes how runtime or memory grows with input. Constant O(1), linear O(n), logarithmic O(log n), quadratic O(n²). Prefer lower growth for scalability.
9

Count vowels in a string.

Answer: Iterate the string and increment a counter when char is in vowel set (a,e,i,o,u). O(n) time, O(1) extra space.
10

Remove duplicates from an unsorted array preserving order.

Answer: Use a seen set and iterate list, appending unseen items to result; preserves first occurrence. O(n) time, O(n) space.

Scenario 1: Reverse a String

Interviewer:

Write a function to reverse a string.

Candidate:

def reverse_string(s):
    return s[::-1]

Interviewer:

What is the time complexity?

Candidate:

O(n), where n is the length of the string.

Scenario 2: Binary Search

Interviewer:

Implement binary search.

Candidate:

def binary_search(arr, target):
    left, right = 0, len(arr)-1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

Complexity: O(log n).

Scenario 3: Palindrome Check

Interviewer:

Check if a string is a palindrome.

Candidate:

def is_palindrome(s):
    return s == s[::-1]

Edge Cases: Empty string, single character.
